# Grupo 2 - VClothes
### Producto: 
En éste proyecto vamos a crear una página web sencilla y minimalista para la venta de ropa estilo vintage.
### Público:
Nuestro público objetivo son adolecentes y adultos que les guste vestir con éste estilo.
